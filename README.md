#📱 Calculadora de Bitola de Fios
## 📌 Descrição do projeto

Este projeto é um aplicativo mobile desenvolvido em Dart utilizando o Flutter que calcula a bitola adequada de fios elétricos para instalações residenciais.

O usuário pode inserir:

* Corrente elétrica em ampères
* Distância do circuito em metros

A partir desses dados, o aplicativo calcula:

* Bitola do fio em milímetros quadrados para tensão de 110V
* Bitola do fio em milímetros quadrados para tensão de 220V

O objetivo é auxiliar na escolha correta da bitola dos cabos elétricos.
